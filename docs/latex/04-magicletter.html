特殊な文字の扱い (%, &, $, #, \, _, {, })

情報リテラシ第一 / 情報リテラシ第二

いくつかの文字はpLaTeXのなかで特殊な扱いを受けます。

たとえば%はコメントの始まりを表す記号です。ですので、文章中に%と書くと%とその行の%の後ろに続く部分は出力されません。

同様に$は数式モードの表現に、&は表や連立方程式の書式に、~は明示的な一文字分の空白を表します。これらの特殊な文字を入力するには特別な配慮を要します。

出力したい文字 % & $ # \ _ { }
pLaTeXの記法 \% \& \$ \# $\backslash$ \_ \{ \}

ここの記述は熊澤先生の解説ページを参考にして書きました。
